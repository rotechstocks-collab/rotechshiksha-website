## Usage Guidelines

1. **Placement:** Use dialogues at section headers or feature introductions
2. **Animation:** Fade in dialogue bubbles for engagement
3. **A/B Test:** Test different dialogues for same page
4. **Mobile:** Keep 2-line dialogues for mobile, 4-line for desktop
5. **Context:** Match dialogue to user's likely mindset on that page
